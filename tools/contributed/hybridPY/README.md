# hybridPY
### Overview
hybridPY is intended to expand the user-base of the traffic micro-simulator SUMO (Simulation of Urban MObility) 
an the mesoscopic simulator MATSim by providing a user-friendly, yet flexible simulation suite.

A further scope of hybridPY is to manage the huge amount of data necessary to run complex multi-modal simulations.
This includes different demand generation models as well as a large range of modes, such as public transport, shared autonomous vehicle (SAV) sevice, private taxis,
bicycles and personal rapid transit (PRT). hybridPY consists of a GUI interface, network editor as well as a simple to use scripting language which facilitates the use of SUMO and MATSim.

This is the software stack for hybridPY. It is based on https://github.com/schwoz/sumopy
